What is important

  • デザイン

    お客様の声の中から
    ご要望・問題点を洗い出し、解決に向けヒアリングを重ねながらプランを作成します。
    決まったプランに沿って使いやすさだけでなく楽しさや心地よさといった UI・UXの考えに基づいたデザインを導き出します。

  • コーディング

    HTML・CSSでデザインを忠実に再現するだけでなく、JavaScript・JQueryを使用し動きのあるイメージをプラスすることでワンランク上のデザインを提供します。

  • マーケティング

    デザインのブラッシュアップの精度を高める為、お客様がご活躍されているフィールド（業種・業態）についての情報収集を行いアンテナを張り巡らせることを常に大切にしています。
